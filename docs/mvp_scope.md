# MVP Scope — Digital Concept Quote Builder v2

## Goal

Create an internal sales-facing Streamlit app that helps PFM move from customer intake to a reviewed concept quote preview.

## In scope

- Retail Chain only.
- Sales intake form.
- Performance need / pain selection.
- PFM fit mapping.
- Retail ROI/TCO model.
- Essential / Professional / Enterprise suggested route.
- Concept quote preview.
- Mock Odoo payload preview.
- Mock n8n payload preview.
- JSON download for internal review.
- Human review checklist.

## Out of scope

- Shopping Centre quoting.
- Public lead-generation tool.
- Production Odoo writes.
- Live n8n workflows.
- Automatic customer emails.
- Final PDF quotation generation.
- Production salesperson IDs, template IDs or pricelists.
- Unapproved discounts.
- ANPR, Re-ID, geo/catchment and advanced privacy-sensitive modules.

## MVP success criteria

The MVP is successful when an internal salesperson can:

1. enter basic customer details;
2. select customer needs;
3. see an indicative ROI/TCO outcome;
4. see a suggested route and concept quote preview;
5. inspect mock Odoo/n8n payloads;
6. download a JSON payload for internal review;
7. clearly understand that human review is required before customer use.
